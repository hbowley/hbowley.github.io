Shiny

Store Performance Tracking System

Built for Abcaco Foods through Portable Networks — a production Shiny application deployed on AWS with a PostgreSQL backend that enables real-time sales data entry for field teams.

What it does:

  • Real-time sales data entry and tracking for field representatives
  • Product placement monitoring across store locations
  • Manager engagement dashboards
  • Live analytics on store performance trends

Architecture:

  • Shiny frontend deployed on AWS EC2
  • PostgreSQL (RDS) backend for persistent data storage
  • Role-based access for field teams vs. management
  • Automated reporting and export functionality

Tech: R, Shiny, AWS (EC2, RDS), PostgreSQL, Production Deployment

Workforce BI Platform

Built for Lucto Fire through Portable Networks — a Shiny-based business intelligence application that integrates Salesforce Reports to provide real-time operational visibility.

What it does:

  • Real-time tracking of active projects pulled from Salesforce
  • Forecasted workforce demand and contractor allocation
  • Resource planning dashboards for timely hiring decisions
  • Executive-level reporting on project pipeline health

Tech: R, Shiny, Salesforce API, Business Intelligence, Forecasting
