# Hannah Bowley Portfolio

Personal portfolio site built with Hugo.

## Edit content

- Homepage: `themes/minimal/layouts/index.html`
- About: `content/about.md`
- Resume page: `content/resume.md`
- Projects: `content/projects/*.md`
- Styles: `static/css/style.css`
- Static files (PDF/images): `static/`

## Deploy

This repo deploys via GitHub Actions on push to `main`.

## Notes

- Keep copy direct and concise.
- Avoid em dashes in content.
