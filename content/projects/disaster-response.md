Built a Python and R-based automated pipeline using Selenium and Google Sheets API to track utility outages, extreme heat, and boil water notices during Hurricane Milton for Catalyst Miami / CEFA-Clean Energy for All.

**Impact:**
- Centralized real-time outage data used to activate community response teams
- Enhanced disaster relief coordination across Miami-Dade County
- Scalable framework for future emergency response deployments

**How it works:**
- Selenium scrapes live utility outage data at regular intervals
- Data flows through Google Sheets API for real-time team access
- R scripts generate summary analytics and trend reports
- Automated alerts when outage thresholds are exceeded

**Tech:** Python, R, Selenium, Google Sheets API, Automation
